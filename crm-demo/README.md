# CRM Performance — Демо (только фронт)

Полноценный клон интерфейса вашей CRM в демо-режиме: те же страницы, боковое меню, дашборд, аналитика. Без бэкенда — все данные моковые.

## Запуск

```bash
cd crm-demo
npm install
npm run dev
```

Откройте в браузере: **http://localhost:5174**

По умолчанию вы «залогинены» как staff, доступны все разделы.

## Сборка

```bash
npm run build
npm run preview   # просмотр собранной версии на порту 5174
```

## Как устроено

- Исходный код UI берётся из основной папки `src` (общий с основной CRM).
- В `crm-demo/src/demo/` подменены только:
  - **useAuth** — всегда возвращает демо-пользователя (staff).
  - **auth.service** — без запросов к серверу.
  - **api** — все вызовы возвращают мок-данные.

Остальные компоненты, страницы и стили — те же, что в полной CRM.
