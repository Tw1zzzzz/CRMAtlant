# CRM Performance Demo Page

Отдельная тестовая страница именно для текущей CRM Performance.

Демо повторяет структуру вашей системы:

- левая навигация с разделами (`Обзор`, `Настроение и Энергия`, `Тесты`, `Статистика`, `Аналитика`, `Колесо баланса`, `Топ игроков`, `Файловое хранилище`);
- тёмная палитра из проекта (`#111827`, `#1A202C`, `#3590FF`, `#00E396`);
- блоки `Mood / Energy`, активность, топ игроков.

## Файлы

- `index.html` - демо интерфейс CRM Performance
- `styles.css` - оформление и адаптив
- `script.js` - часы, KPI-счетчики, переключение данных графика

## Запуск

**Вариант 1 — через npm (рекомендуется):**

```bash
cd crm-demo-frontend
npm install
npm start
```

**Вариант 2 — через Python:**

```bash
cd crm-demo-frontend
python3 -m http.server 4173
```

Открыть в браузере: **http://localhost:4173**
